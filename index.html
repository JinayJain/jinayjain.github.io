Projects

  • Plorium

    Plorium aims to drastically simplify the process of self-learning a new topic on the internet. Through a community-curated list of content, Plorium will help you find the best learning resources quickly and efficiently, eliminating the need for constant searching.

    I am currently building Plorium with React, Next.js, PostgreSQL, and GraphQL.

  • Just a Minute

    Just a Minute is a web experiment designed to gauge the perception of time, simply by estimating the length of a minute. With over 100,000 players, the project went viral and was featured on the front page of Hacker News.

    I built Just a Minute using HTML/CSS/JS and Firebase. Read more about my experience in my blog post.

  • Backpropagation Visualizer

    I built a visualizer that demonstrates how backpropgation works, and how it can be used to optimize a neural network. The visualizer parses math expressions and automatically generates a computation graph from it. Finally, it automatically differentiates the math expression and shows the gradient at each node in the graph.

    To help others understand backpropagation, I wrote a blog post with a more detailed explanation.

  • TFCP

    TFCP is a code generation AI that harnesses the large code repositories of competitive programmers and trains a recurrent neural network on their code. Using Tensorflow and Keras, I created a Python script that would learn someone's coding style and generate similar pieces of "code" that mimics their way of programming.

    Note: This project was created in 2019, before GitHub Copilot and AlphaCode, so I guess they owe me some licensing fees.

  • Vim Docs

    Vim Docs is a realtime collaborative web editor that uses basic keybindings from the vim text editor and collaboration like Google Docs. I built the site using a Vue frontend and an Express.js backend built with Node.js. I used Heroku to deploy the website and maintain it.
